git拉取权限拒绝

发表于 2017-08-03 | 分类于 运维工具 ， git | | 阅读数

自己挖的坑…

由于昨晚在家新增和修改了文件,今早到公司就进行了个pull的操作

全是权限不足,第一反应是sudo的原因.没多想就直接sudo操作了,然后提示

搜索

解决办法都是说重新生成秘钥.然后复制到git.
但是自己的密钥并没有变更过,以防万一,还是操作了一遍.

并没有解决问题.

想想如果不是远程的问题,那就是本地的了.

原来昨晚创建了新的文件夹用的是sudo创建的…

更改文件夹所属就好了
